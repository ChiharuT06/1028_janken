# 課題　 --
- エモい文章をランダム生成するアプリ
## ①課題内容（どんな作品か）
- いくつかのボタンを押すと良い感じにエモい文章を作ってくれるアプリ 

## ②工夫した点・こだわった点
- 文章だけでなく背景やフォントでもエモさを演出
- back-groundを半透明にしても文字はくっきり見えるようにコードを書いた

## ③難しかった点・次回トライしたいこと(又は機能)
- display:noneの表示の切り替えに苦労した
- ボタンのデザインを多用なものにしたかった
